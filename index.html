Приложение для управления и аналитики каталога женских сумочек с полным набором функций для работы с данными. Возможности: создание новых записей, удаление ненужных, редактирование существующих, расширенная фильтрация по различным параметрам, сортировка по любому полю, загрузка данных из Excel, экспорт данных в Excel, сохранение изменений обратно в таблицу.
